# 🚦 Real-Time Traffic Streaming Control Room

A professional real-time traffic monitoring and analytics dashboard built using **Parallel Computing concepts**, **multithreading**, and **Streamlit**.

This project simulates a smart traffic control room where multiple worker threads process live traffic sensor data simultaneously using a **Producer–Consumer Architecture**.
